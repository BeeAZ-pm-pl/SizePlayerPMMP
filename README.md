# SizePlayerPMMP
A PocketMine-MP plugin that allows players to change their size.

# [ Commands ]
+ /size <number>
+ /size help
+ /size reset
  
# [ What's New ]
+ [x] Decreased a maximum sizes from 20 to 15.
+ [x] Added an icon.
